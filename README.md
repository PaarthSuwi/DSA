# 📊 DSA — Data Structures & Algorithms

A growing collection of **Data Structures and Algorithms** implementations in C++. Built as part of a consistent daily practice routine to strengthen problem-solving and competitive programming skills.

## 📚 Topics Covered

| Category | Topics |
|----------|--------|
| **Sorting** | Bubble Sort, Selection Sort, Insertion Sort, Merge Sort, Quick Sort |
| **Searching** | Linear Search, Binary Search |
| **Data Structures** | Arrays, Linked Lists, Stacks, Queues, Trees, Graphs |
| **Dynamic Programming** | Memoization, Tabulation |
| **Graph Algorithms** | BFS, DFS, Dijkstra's |
| **Recursion** | Backtracking, Divide & Conquer |
